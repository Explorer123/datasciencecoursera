####Part 3 Imputing Missing Values
-num_na:Number of missing values (NA)

Strategy
-Merged original data set with interval means data frame (agg_mean_ti) to replace NA steps for an interval with mean for that interval. -assigned the mean to the NA values for that interval in the new merged dataframe -dropped the merged “means” column to have the same number of columns as original data set but with NAs replaced with the mean. New data frame is a1_new

-plot histogram of total number of steps each day (new data frame, a1_new has no NA for steps)

new_mean: New mean after imputing missing values new_median: New median after imputing missing values new_sum:New median after imputing missing values

Effect on mean: No difference
Effect on median: Yes, those that showed as NA for orginal, new had a value. Others remained same
Effect on total number of steps: Yes, sum had higher values for the those dates that had NA before. Other dates that did not have NA before had no effect in the new data frame
